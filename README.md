### `<base_ver>+prefixed`

All commands are prefixed with `wps` to prevent conflicts (e.g. https://github.com/MisterTea/EternalTerminal/issues/316).

| Original | Prefixed |
|----------|----------|
| `et`     | `wpset`  |
| `wpp`    | `wpswpp` |
| `wps`    | `wps`    |
| `wpspdf` | `wpspdf` |
